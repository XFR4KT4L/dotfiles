# Dotfiles — Arch Linux + Hyprland

## Hardware
- GPU: NVIDIA RTX 3070
- CPU: Intel
- RAM: 32GB DDR5
- Monitor 1: DP-3 — 4K 60Hz (izquierda, desarrollo)
- Monitor 2: DP-4 — 2K 360Hz (derecha, gaming)

## Stack
- WM: Hyprland (Wayland)
- Bar: Waybar (Catppuccin Mocha pill)
- Terminal: Kitty
- Shell: Zsh + Starship
- Launcher: Wofi
- Notificaciones: Mako
- Tema: Catppuccin Mocha Mauve
- Iconos: Papirus Dark
- Cursor: Bibata Modern Classic
- Navegador: Zen Browser
- Editores: Neovim + VSCode

## Instalar en máquina nueva
```bash
git clone https://github.com/TU_USUARIO/dotfiles ~/.dotfiles
cd ~/.dotfiles && chmod +x setup.sh && ./setup.sh
```
